# Meteor 部署的几种方式

## Windows 部署方式
### Mongodb 设置
启动Mongodb服务器<br>
```c:\mongod```<br>
shell登陆Mongodb服务器<br>
```c:\mongo```<br>
添加管理用户及权限，Meteor使用此帐号连接Mongodb服务器<br>
```\> db.createUser({user:’admin’,pwd:’password’,roles:[‘readWrite’,’dbAdmin’]})```<br>
